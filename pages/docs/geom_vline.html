Usage

geom_vline(position = 'identity', stat = 'vline')

Arguments

Name Description
position Position adjustments to points. identity
stat he statistical transformation to use on the data for this layer. vline

Aesthetics

  • xintercept
  • linetype
  • color
  • alpha
ggplot(mtcars, aes(x='wt', y='mpg')) +\
    geom_point() +\
    geom_vline(xintercept=[2, 4, 6])

ggplot(mtcars, aes(x='wt', y='mpg')) +\
    geom_point() +\
    geom_vline(xintercept=[2, 4, 6], linetype='dashed')
