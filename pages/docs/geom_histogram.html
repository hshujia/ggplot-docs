Usage

geom_histogram(position = 'stack', stat = 'bin')

Arguments

Name Description
position Position adjustments to points. stack
stat he statistical transformation to use on the data for this layer. bin | identity

Aesthetics

  • x
  • alpha
  • colour
  • fill
  • linetype
  • size
  • weight
ggplot(aes(x='pageviews'), data=pageviews) + \
    geom_histogram()

ggplot(aes(x='pageviews'), data=pageviews) + \
    geom_histogram(binwidth=50)

ggplot(aes(x='pageviews'), data=pageviews) + \
    geom_histogram(binwidth=1000)

import pandas as pd
import numpy as np

df = pd.DataFrame({
    "x": np.random.normal(0, 10, 1000),
    "y": np.random.normal(0, 10, 1000),
    "z": np.random.normal(0, 10, 1000)
})
df = pd.melt(df)

ggplot(aes(x='value', color='variable'), data=df) + \
    geom_histogram()

ggplot(aes(x='value', fill='variable', color='variable'), data=df) + \
    geom_histogram(alpha=0.6)
