Usage

geom_jitter(position = 'jitter', stat = 'identity')

Arguments

Name Description
position Position adjustments to points. jitter
stat The statistical transformation to use on the data for this layer. identity

Aesthetics

  • x
  • y
  • size
  • shape
  • color
  • alpha
ggplot(diamonds, aes(x='carat', y='price', color='color')) +\
    geom_point(alpha=0.1)

ggplot(diamonds, aes(x='carat', y='price', color='color')) +\
    geom_jitter(alpha=0.1)

ggplot(mtcars, aes(x='wt', y='mpg')) + \
    geom_point()
