title: scale_y_log {% extends "docpage.html" %} {% block body %}

Usage

scale_y_log(name = ..., breaks = ..., labels = ..., limits = ...)

Arguments

Name Description
name Label for y axis.
breaks Points at which y gridlines appear.
labels Labels for y ticks.
limits Where y axis starts/stops.

Examples

import pandas as np
import numpy as np
df = pd.DataFrame({"x": np.arange(100)})
df['y'] = df.x**2

ggplot(aes(x='x', y='y'), data=df) + geom_line()
ggplot(aes(x='x', y='y'), data=df) +\
    geom_line() +\
    scale_y_log(base=2)
