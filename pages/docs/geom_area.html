Usage

geom_area(position = 'stack', stat = 'identity')

Arguments

Name Description
position Position adjustments to points. stack
stat The statistical transformation to use on the data for this layer. identity

Aesthetics

  • x
  • ymin
  • ymax
  • alpha
  • fill
  • color
  • alpha
  • linetype
ggplot(aes(x='date', ymin='beef - 1000', ymax='beef + 1000'), data=meat) + \
    geom_area()

ggplot(aes(x='date', y='beef', ymin='beef - 1000', ymax='beef + 1000'), data=meat) + \
    geom_area() + \
    geom_point(color='coral')
